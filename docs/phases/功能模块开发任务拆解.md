### Domain 编写规则

- 继承 `com.aiplatform.common.core.domain.BaseEntity`（自动获得 `createTime`/`updateTime`/`createBy`/`updateBy`/`remark`）
- 字段名使用 `lowerCamelCase`，与数据库列名 `snake_case` 对应（MyBatis `mapUnderscoreToCamelCase=true` 自动映射）
- `isDelete` 字段单独声明（不走 BaseEntity），用于逻辑删除

### Mapper 接口方法规范

每个 Mapper 至少包含以下方法：
```java
List<Xxx> selectXxxList(Xxx xxx);       // 条件分页查询（配合 PageHelper）
Xxx selectXxxById(Long id);             // 根据 ID 查询
int insertXxx(Xxx xxx);                 // 新增
int updateXxx(Xxx xxx);                 // 修改
int deleteXxxByIds(Long[] ids);         // 批量删除（逻辑删除，更新 is_delete = 1）
```

### AiChatServiceImpl.chat() 核心流程

```
chat(dto):
  1. 校验 dto.message not blank
  2. 根据 conversationId 获取或创建 AiConversation
     - 有 id: AiConversationMapper.selectById()
     - 无 id: 新建会话（userId=当前登录用户, title=截取message前30字, agentType=dto.agentType or "planner"）
  3. 保存用户消息: AiMessageMapper.insert(role="user", content=message)
  4. 调用 Agent: AgentClient.execute(task="chat", input=buildAgentRequest(dto))
     → M5 完成后改为 FastApiClient.post("/api/v1/chat", ...)
     → 当前阶段用 Mock: return "这是模拟回复：" + message
  5. 解析 Agent 返回: 提取 content/agentType/tokenUsage/sources
  6. 保存 AI 回复: AiMessageMapper.insert(role="assistant", content=..., sourceType=agentType)
  7. 更新会话: conversationMapper.update(title=自动截取, updateTime=now)
  8. 返回 ChatResponseDto
```

### DocumentServiceImpl.uploadDocument() 核心流程

```
uploadDocument(kbId, file, chunkSize):
  1. 校验文件类型（txt/pdf/md/docx/xlsx/html）
  2. 保存文件到磁盘: backend/upload/ai/kb/{kbId}/
  3. 提取文本:
     - txt/md: 直接读取
     - pdf: Tika/PDFBox 提取
     - docx: POI 提取
     - 其他: Apache Tika
  4. 插入文档记录: KbDocumentMapper.insert(status=0, filePath=, contentText=)
  5. 文本分段:
     - 按段落分割（\n\n）
     - 超长段落按 chunkSize 切分（滑动窗口重叠 chunkOverlap）
  6. 更新文档: status=1(处理中), chunkCount=分段数
  7. 调用向量索引（暂 Mock）:
     → M6 完成后调用 IndexService.indexDocument(doc)
     → 当前阶段: 记录日志 "待向量化: docId=X, chunkCount=Y"
  8. 更新文档: status=2(完成)
  9. 更新知识库: kbKnowledge.docCount += 1
  10. 返回 KbDocument
```

### FastApiClient 核心方法

```java
@Component
public class FastApiClient {

    @Autowired
    private WebClient.Builder webClientBuilder;

    @Value("${ai.agent.base-url}")
    private String agentBaseUrl;

    @Value("${ai.agent.api-prefix:/api/v1}")
    private String apiPrefix;

    /**
     * 同步 POST
     */
    public String post(String path, Map<String, Object> body) {
        return webClientBuilder.build()
            .post()
            .uri(agentBaseUrl + apiPrefix + path)
            .bodyValue(body)
            .retrieve()
            .onStatus(HttpStatusCode::isError, resp ->
                resp.bodyToMono(String.class)
                    .flatMap(err -> Mono.error(new FastApiException(resp.statusCode().value(), err))))
            .bodyToMono(String.class)
            .block(Duration.ofSeconds(300));
    }

    /**
     * 流式 POST（SSE 接收 Python Agent 流式响应）
     */
    public Flux<String> postStream(String path, Map<String, Object> body) { ... }

    /**
     * GET
     */
    public String get(String path) { ... }
}
```

### chatStream() 核心流程

```
chatStream(dto):
  1. 创建 SseEmitter(300_000L)
  2. 获取/创建会话，保存用户消息（同 chat()）
  3. CompletableFuture.runAsync(() -> {
       try {
         FastApiClient.postStream("/api/v1/chat/stream", body,
           chunk -> emitter.send(SseEmitter.event().name("message").data(chunk))
         );
         // 收集全部 delta 拼接完整消息 → 保存到 AiMessage
         saveMessage(convId, "assistant", fullContent);
         emitter.send(SseEmitter.event().name("message")
             .data("{\"type\":\"end\",\"messageId\":" + msgId + "}"));
         emitter.complete();
       } catch (Exception e) {
         emitter.send(SseEmitter.event().name("error")
             .data("{\"type\":\"error\",\"message\":\"" + e.getMessage() + "\"}"));
         emitter.completeWithError(e);
       }
     });
  4. 注册 emitter 到停止管理器（便于 stopGeneration 调用）
  5. 返回 emitter
```

### SSE 事件协议

| 事件名 | 数据结构 | 时机 |
|--------|---------|------|
| `message` | `{"type":"start","conversationId":123,"messageId":456}` | 流开始 |
| `message` | `{"type":"delta","content":"Spring"}` | 逐块文本 |
| `message` | `{"type":"tool_call","toolName":"search_kb","args":{...}}` | Agent 调用工具 |
| `message` | `{"type":"tool_result","toolName":"search_kb","result":"..."}` | 工具返回结果 |
| `message` | `{"type":"end","messageId":456,"tokenUsage":{...}}` | 流结束 |
| `error` | `{"type":"error","message":"调用超时"}` | 异常中断 |

### 前端 API 调用示例

```typescript
// frontend/src/api/ai/chat.ts
import request from '@/utils/request';

export function chat(data: ChatRequest) {
  return request({ url: '/ai/chat', method: 'post', data });
}

// SSE 流式
export function chatStream(data: ChatRequest, onDelta: (text: string) => void, onDone: () => void) {
  const token = getToken();
  return fetch('/dev-api/ai/chat/stream', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', Authorization: `Bearer ${token}` },
    body: JSON.stringify(data),
  }).then(async (response) => {
    const reader = response.body!.getReader();
    const decoder = new TextDecoder();
    let buffer = '';
    while (true) {
      const { done, value } = await reader.read();
      if (done) { onDone(); break; }
      buffer += decoder.decode(value, { stream: true });
      const lines = buffer.split('\n');
      buffer = lines.pop() || '';
      for (const line of lines) {
        if (line.startsWith('data: ')) {
          const event = JSON.parse(line.slice(6));
          if (event.type === 'delta') onDelta(event.content);
          if (event.type === 'end') onDone();
        }
      }
    }
  });
}
```
